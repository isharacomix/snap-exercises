TASK: Draw a triangle

The triangle shape in the window is called a sprite. If you use the 'pen' block, then whenever the sprite moves, it will draw a line behind it.

  • Use the 'pen down' block to start drawing
  • Use the 'move' block to move the arrow in the direction it's pointing.
  • Use the 'turn' block to rotate the arrow (remember, there are 360 degrees in a circle)

When you're done, check your answer. Your answer might be different from the one we provide, but that's OK as long as you're drawing a triangle.
